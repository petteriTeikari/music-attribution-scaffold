# STYLE GUIDE — Frontend Hero & Feature Images

**Version:** 1.0.0
**Scope:** Frontend hero images, feature illustrations, waveform art
**Target:** Bold editorial art — Warp Records aesthetic meets constructivist data viz

---

## DESIGN AESTHETIC

### Target Style

- **50%** Warp Records CD artwork — abstract waveform visualization, Roman numerals, pale backgrounds
- **25%** Constructivist data art — geometric compositions in red/black/blue on cream
- **25%** Scandinavian editorial — dramatic whitespace, Bauhaus-inspired bold shapes

### Visual Identity

Frontend figures communicate **bold artistry, technical precision, and warm editorial quality**:

- Bold: Coral red accent as primary graphic element
- Precise: Data visualization as art, confidence as visual language
- Warm: Cream backgrounds, editorial photography feel, no cold/corporate

### CRITICAL: NO AI SLOP

**BANNED:**
- Generic stock art or photography
- Corporate SaaS gradients
- Sci-fi neon/glow/holographic effects
- Symmetric, predictable compositions
- Rounded-everything pill aesthetics
- Dark-mode-first design
- Literal screenshots of the UI
- Oversaturated colors

**REQUIRED:**
- Abstract interpretations of data concepts
- Bold geometric compositions
- Warm cream (#f6f3e6) as primary background
- Coral red (#E84C4F) as primary accent
- Asymmetric, editorial layouts
- Halftone or grain textures where appropriate

---

## COMPOSITION RULES

1. **Asymmetric grids** — never perfectly centered or symmetrical
2. **Accent squares** — 28px solid coral squares as graphic punctuation
3. **Thin accent lines** — 1px coral horizontal rules as section dividers
4. **Roman numerals** — I, II, III... for sequential elements (Warp Records homage)
5. **Negative space** — at least 30% of canvas should be empty
6. **Overlapping elements** — text can overlap into graphic space
7. **No drop shadows** — flat, matte finishes only
